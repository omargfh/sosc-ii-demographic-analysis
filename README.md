## Demographic Analysis

This repository contains the code and data for class SOSC 16300 Global Society III at the University of Chicago. The goal of this project is to analyze the mortality crisis that followed the collapse of the Soviet Union in terms of population, correlational, and sentiment data. The project is divided into three parts: a demographic analysis, a correlation analysis, and a sentiment analysis.

### Data
* [hash_rusentilex_2017: General Russian Sentiment Lexicon RuSentiLex 2017](https://rdrr.io/github/dmafanasyev/rulexicon/man/hash_rusentilex_2017.html)
* [WHO Mortality Database](https://www.who.int/healthinfo/statistics/mortality_rawdata/en/)
* [WHO EWP Database](https://www.who.int/healthinfo/statistics/ewp/en/)
* [WHO EMD Database](https://www.who.int/healthinfo/statistics/emd/en/)
* [UNData](http://data.un.org/)
